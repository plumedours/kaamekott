# 🏰 Kaamekott

Un site dédié aux meilleures citations de Kaamelott, enrichi avec des GIF pour les moments où les mots ne suffisent plus.

👉 https://plumedours.github.io/kaamekott/

---

## ✨ Présentation

Kaamekott est une application web permettant de :

- 📜 Parcourir les citations de Kaamelott
- 👤 Naviguer par personnage
- 📚 Explorer par livre / saison / épisode
- 🎞️ Rechercher et utiliser des GIF
- 🎲 Obtenir une citation et un GIF aléatoires
- 🔍 Retrouver rapidement une réplique via la recherche

Le tout avec une interface moderne, rapide et pensée pour être agréable à utiliser.

---

## 🚀 Fonctionnalités principales

### 📖 Citations
- Liste complète des citations
- Filtrage par personnage
- Navigation par livre et épisode
- Vue détaillée par épisode

### 🎞️ GIF
- Galerie complète de GIF
- Recherche par texte, personnage ou épisode
- Page dédiée par GIF
- Copie de lien et partage

### 🎲 Aléatoire
- Génération d’une citation + GIF aléatoire
- Bouton pour relancer instantanément

### 🧠 UX
- Interface responsive
- Navigation fluide
- Ton inspiré de l’univers Kaamelott

---

## 🛠️ Stack technique

- ⚛️ React
- ⚡ Vite
- 🎨 TailwindCSS
- 🧭 React Router (HashRouter)
- 🚀 GitHub Pages (déploiement automatique via GitHub Actions)

---

## 📦 Installation

```bash
git clone git@github.com:plumedours/kaamekott.git
cd kaamekott
npm install
npm run dev
```

---

## 🏗️ Build

```bash
npm run build
```

---

## 🌍 Déploiement

Le projet est automatiquement déployé via GitHub Actions à chaque push sur la branche `main`.

Configuration :
- Build Vite → dossier `dist`
- Déploiement via GitHub Pages
